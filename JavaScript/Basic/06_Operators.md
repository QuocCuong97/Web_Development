# Biểu thức và toán tử
## **1) Các biểu thức toán học**
| Toán tử | Mô tả |
|---------|-------|
| `+` | Cộng |
| `-` | Trừ |
| `*` | Nhân |
| `**` | Lũy thừa |
| `/` | Chia |
| `%` | Chia (lấy phần dư) |
| `++` | Tăng dần |
| `--` | Giảm dần |
- **VD :**
    - **Cộng** :
        ```js
        var x = 5;
        var y = 2;
        var z = x + y;
        ```
        - Kết quả : `z = 7`
    - **Trừ** :
        ```js
        var x = 5;
        var y = 2;
        var z = x - y;
        ```
        - Kết quả : `z = 3`
    - **Nhân** :
        ```js
        var x = 5;
        var y = 2;
        var z = x * y;
        ```
        - Kết quả : `z = 10`
    - **Lũy thừa** :
        ```js
        var x = 5;
        var z = x ** 2;
        ```
        - Kết quả : `z = 25`
    - **Chia** :
        ```js
        var x = 5;
        var y = 2;
        var z = x / y;
        ```
        - Kết quả : `z = 2.5`
    - **Chia lấy phần dư** :
        ```js
        var x = 5;
        var y = 2;
        var z = x % y;
        ```
        - Kết quả : `z = 2`
    - **Tăng dần** :
        ```js
        var x = 5;
        x++;
        var z = x;
        ```
        - Kết quả : `z = 6`
    - **Giảm dần** :
        ```js
        var x = 5;
        x--;
        var z = x;
        ```
        - Kết quả : `z = 4`
    - **Biểu thức chứa nhiều toán tử** :
        ```js
        var x = 100 + 50 * 3;
        ```
        - Kết quả : `x = 250`
